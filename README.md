    Calculator that moves on terminal using C++.

## About the Project
Console game with C++.
Player will either choose from rock paper scissors game or number guessing game.
Program will run with English or Japanese.

- c++17
- Terminal

## How to Build (Linux/Mac)
1.Clone the repository to your local enviroment.

2.Move to directory that contains the app on terminal.

3.Build program on your terminal with command below:
```bash
    make
```

4.Run program with command below:
```bash
    ./main
```

## How to Use
1.Input number and preffered language between English or Japanese.

2.Input number and choose game to play.

3.For rock paper scissors game, choose number that matches your hand that you chose and input. Computer will randamly choose their hand.

4.For number guessing game, guess and input number that computer randamly chose between 1-100.

5.Choose "Y" to play the game again, "M" to change game, "N" to finish the application.
