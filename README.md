# Terminal game Application
  Game application that moves on terminal using C++.
    
## Table of Contents
- About the Project
- Environment
- Directory
- How to Build
- How to Use

## About the Project
Players can choose between two games:
- Rock Paper Scissors
- Number Guessing Game

The application supports English or Japanese.

## Environment
- c++17
- Terminal

## How to Build (Linux/Mac)
1.Clone the repository to your local enviroment.

2.Move to directory that contains the app on terminal.

3.Build program on your terminal with command below:
```bash
    make
```

4.Run program with command below:
```bash
    ./main
```

## How to Use
1.Input number and preffered language between English or Japanese.

2.Input number and choose game to play.

3.If you select rock paper scissors game, enter the number corresponding to your chosen hand. Computer will randamly choose it's hand.

4.If you select number guessing game, guess and input number that randamly generated by the computer between 1-100.

5.Choose "Y" to play the game again, "M" to change game, "N" to finish the application.
